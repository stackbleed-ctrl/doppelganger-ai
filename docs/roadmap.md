# Doppelganger Roadmap

## v0.1.0 — Foundation (current)
- [x] Event bus + 5-layer architecture
- [x] Grok-3 integration
- [x] Parallel world simulation
- [x] Temporal memory graph (JSON-backed)
- [x] Qdrant vector search
- [x] faster-whisper STT + Kokoro TTS
- [x] WiFi CSI sensing (with graceful fallback)
- [x] Skills marketplace (3 built-in skills)
- [x] React dashboard
- [x] Docker Compose one-command install
- [x] REST API + WebSocket + SSE streaming
- [x] CLI

## v0.2.0 — Memory upgrade
- [ ] Full Graphiti integration (Neo4j-backed temporal KG)
- [ ] Memory compression and summarization
- [ ] Entity extraction from conversations
- [ ] Memory visualization (force-directed graph in UI)
- [ ] Import from Obsidian, Notion, browser history

## v0.3.0 — Perception upgrade
- [ ] Real nexmon_csi integration tested on RPi4
- [ ] Pose estimation from CSI (walking direction, rough body position)
- [ ] Typing cadence stress estimation
- [ ] Calendar integration for context-aware responses

## v0.4.0 — Voice upgrade
- [ ] Wake word detection ("Hey Doppelganger")
- [ ] Speaker diarization (multi-person household)
- [ ] Emotion detection from voice prosody
- [ ] Multilingual support

## v0.5.0 — Persona system
- [ ] Multiple personas (work / personal / focus)
- [ ] Persona auto-switching based on context
- [ ] Different voice + personality per persona
- [ ] Persona import/export

## v1.0.0 — Stable release
- [ ] Tauri desktop app (no Docker needed)
- [ ] Plugin SDK + public skill registry
- [ ] iOS/Android companion (LAN only)
- [ ] End-to-end tests
- [ ] Performance benchmarks
- [ ] Security audit
